## Run Project

### 1. Tạo database

Mở Azure Data Studio hoặc SQL Server Management Studio
Sau đó chạy file:

database/database.sql


### 2. Chạy project
Nếu SQL Server trên máy của bạn không chạy trên `localhost`,
hãy chỉnh sửa lại **connection string** trong file `appsettings.json` cho phù hợp với tên server SQL Server trên máy của bạn.


Mở file FastFood.sln bằng Visual Studio hoặc Rider

Nhấn Run (F5)
